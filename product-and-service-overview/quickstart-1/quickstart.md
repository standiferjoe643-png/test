# What Standard Migration Includes

By default, Standard Migration covers all native entities supported by your Source and Target platforms:&#x20;

* **Products**
* **Categories & subcategories**
* **Customers**
* **Orders**
* **Taxes**
* **Manufacturers / Brands / Vendors**
* **Reviews**
* **Blog posts & CMS pages**
* **Coupons & discounts**

It also includes the core configuration options in the migration UI, such as:

* **Entity selection (which data types to migrate).**
* **Order status mapping (Source â†’ Target).**
* **Customer group mapping.**
* **Options such as:**
* **Preserve Order IDs**
* **Generate 301 redirects for SEO**
* **Strip HTML from descriptions**

And it always includes:

* **Demo Migration**
* **Re-Migration**
* **Recent Data Migration**
